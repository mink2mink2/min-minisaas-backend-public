> 형식: `US-[도메인][번호]: [역할]로서 [목표]를 위해 [행동]하고 싶다.`

## AUTH 도메인

### US-A01: Google 소셜 로그인
**사용자**로서 Google 계정을 이용해 별도의 회원가입 없이 앱에 로그인하고 싶다.

- **트리거**: 앱에서 "Google로 로그인" 버튼 탭
- **흐름**: Firebase Google 로그인 → ID Token 획득 → 서버에 전달 → JWT 발급
- **결과**: `access_token`(JWT) 및 사용자 정보 반환
- **관련 API**: `POST /api/v1/auth/login/google`

### US-A02: Kakao 소셜 로그인
**사용자**로서 카카오 계정으로 로그인하여 간편하게 서비스를 이용하고 싶다.

- **트리거**: 앱에서 "카카오로 로그인" 버튼 탭
- **흐름**: Kakao SDK 로그인 → Access Token 획득 → 서버에 전달 → 서버가 카카오 API 검증 → JWT 발급
- **결과**: `access_token`(JWT) 및 사용자 정보 반환
- **관련 API**: `POST /api/v1/auth/login/kakao`

### US-A03: Naver 소셜 로그인
**사용자**로서 네이버 계정으로 로그인하여 간편하게 서비스를 이용하고 싶다.

- **트리거**: 앱에서 "네이버로 로그인" 버튼 탭
- **흐름**: Naver OAuth → Access Token 획득 → 서버 검증 → JWT 발급
- **결과**: `access_token`(JWT) 및 사용자 정보 반환
- **관련 API**: `POST /api/v1/auth/login/naver`

### US-A04: 내 정보 조회
**로그인한 사용자**로서 현재 로그인된 계정 정보(이메일, 닉네임 등)를 확인하고 싶다.

- **트리거**: 앱 프로필 화면 진입
- **결과**: 사용자 ID, 이메일, 닉네임, OAuth 제공자 정보 반환
- **관련 API**: `GET /api/v1/auth/me`

### US-A05: 로그아웃
**로그인한 사용자**로서 현재 기기에서 안전하게 로그아웃하고 싶다.

- **트리거**: 앱 설정에서 로그아웃 탭
- **결과**: 서버에서 해당 FCM 토큰 비활성화, 클라이언트 토큰 삭제
- **관련 API**: `POST /api/v1/auth/logout`

## CHAT 도메인

### US-C01: 1:1 채팅방 생성
**로그인한 사용자**로서 다른 사용자와 1:1 채팅방을 만들어 대화하고 싶다.

- **트리거**: 다른 사용자 프로필에서 "채팅하기" 탭
- **흐름**: 기존 채팅방 확인 → 없으면 새로 생성 → 채팅방 입장
- **결과**: 채팅방 ID 및 상세 정보 반환 (이미 있는 방이면 기존 방 반환)
- **관련 API**: `POST /api/v1/chat/rooms`

### US-C02: 내 채팅방 목록 조회
**로그인한 사용자**로서 내가 참여 중인 모든 채팅방 목록을 최신 메시지 순으로 보고 싶다.

- **트리거**: 앱 채팅 탭 진입
- **결과**: 채팅방 목록 (상대방 정보, 마지막 메시지, 안 읽은 메시지 수 포함)
- **관련 API**: `GET /api/v1/chat/rooms`

### US-C03: 실시간 메시지 송수신
**채팅방에 입장한 사용자**로서 상대방과 실시간으로 메시지를 주고받고 싶다.

- **트리거**: 채팅방 진입 후 메시지 입력
- **흐름**: WebSocket 연결 → 메시지 전송 → 상대방에게 실시간 전달 → 상대방 앱 알림
- **결과**: 메시지가 즉시 상대방 화면에 표시됨
- **관련 API**: `WS /api/v1/chat/ws/rooms/{room_id}`

### US-C04: 메시지 이력 조회
**채팅방에 입장한 사용자**로서 이전에 주고받은 메시지를 스크롤하며 확인하고 싶다.

- **트리거**: 채팅방 진입 또는 위로 스크롤
- **결과**: 페이지네이션 적용된 메시지 목록 (발신자, 내용, 시간 포함)
- **관련 API**: `GET /api/v1/chat/rooms/{room_id}/messages`

### US-C05: HTTP 메시지 전송 (폴백)
**채팅방 사용자**로서 WebSocket 연결이 불안정할 때 HTTP로도 메시지를 전송하고 싶다.

- **관련 API**: `POST /api/v1/chat/rooms/{room_id}/messages`

## BOARD 도메인

### US-B01: 게시글 목록 조회
**사용자**로서 커뮤니티 게시판의 게시글 목록을 카테고리별, 최신/인기 순으로 보고 싶다.

- **트리거**: 앱 게시판 탭 진입
- **결과**: 게시글 목록 (제목, 작성자, 좋아요 수, 댓글 수, 작성일 포함)
- **관련 API**: `GET /api/v1/board/posts`

### US-B02: 게시글 작성
**로그인한 사용자**로서 커뮤니티에 새 게시글을 작성하고 싶다.

- **트리거**: 게시판에서 "글쓰기" 탭
- **결과**: 게시글이 저장되고, 팔로워/구독자에게 알림 발송 (이벤트)
- **관련 API**: `POST /api/v1/board/posts`

### US-B03: 게시글 수정
**게시글 작성자**로서 내가 작성한 게시글 내용을 수정하고 싶다.

- **제약**: 본인 게시글만 수정 가능
- **관련 API**: `PUT /api/v1/board/posts/{id}`

### US-B04: 게시글 삭제
**게시글 작성자 또는 관리자**로서 게시글을 삭제하고 싶다.

- **제약**: 본인 게시글 또는 관리자 권한 필요
- **관련 API**: `DELETE /api/v1/board/posts/{id}`

### US-B05: 게시글 좋아요
**로그인한 사용자**로서 마음에 드는 게시글에 좋아요를 누르고 싶다.

- **결과**: 좋아요 수 증가, 중복 좋아요 방지
- **관련 API**: `POST /api/v1/board/posts/{id}/like`, `DELETE /api/v1/board/posts/{id}/like`

### US-B06: 게시글 북마크
**로그인한 사용자**로서 나중에 다시 보고 싶은 게시글을 북마크하고 싶다.

- **관련 API**: `POST /api/v1/board/posts/{id}/bookmark`, `DELETE /api/v1/board/posts/{id}/bookmark`

### US-B07: 댓글 작성
**로그인한 사용자**로서 게시글에 댓글을 달고 싶다.

- **결과**: 댓글 저장, 게시글 작성자에게 알림 발송 (이벤트)
- **관련 API**: `POST /api/v1/board/posts/{id}/comments`

### US-B08: 댓글 목록 조회
**사용자**로서 게시글의 모든 댓글을 시간순으로 보고 싶다.

- **관련 API**: `GET /api/v1/board/posts/{id}/comments`

### US-B09: 댓글 삭제
**댓글 작성자 또는 관리자**로서 댓글을 삭제하고 싶다.

- **관련 API**: `DELETE /api/v1/board/comments/{id}`

### US-B10: 게시글 검색
**사용자**로서 키워드로 게시글을 검색하고 싶다.

- **관련 API**: `GET /api/v1/board/posts?search={keyword}`

## BLOG 도메인

### US-BL01: 블로그 게시글 작성
**로그인한 사용자**로서 블로그에 새 글을 작성하고 싶다.

- **트리거**: 블로그 탭에서 "새 글 작성"
- **결과**: 블로그 포스트 저장, 구독자에게 알림 발송 (이벤트)
- **관련 API**: `POST /api/v1/blogs`

### US-BL02: 블로그 게시글 수정
**블로그 작성자**로서 내 블로그 글을 수정하고 싶다.

- **관련 API**: `PUT /api/v1/blogs/{id}`

### US-BL03: 블로그 게시글 삭제
**블로그 작성자**로서 내 블로그 글을 삭제하고 싶다.

- **관련 API**: `DELETE /api/v1/blogs/{id}`

### US-BL04: 블로그 피드 조회
**로그인한 사용자**로서 내가 구독한 블로거들의 최신 글을 피드 형태로 보고 싶다.

- **트리거**: 블로그 피드 탭 진입
- **결과**: 구독 작가들의 최신 게시글 목록 (최신순)
- **관련 API**: `GET /api/v1/blogs/feed`

### US-BL05: 블로그 목록 조회
**사용자**로서 모든 블로그 게시글을 카테고리/인기 순으로 탐색하고 싶다.

- **관련 API**: `GET /api/v1/blogs`

### US-BL06: 블로그 좋아요
**로그인한 사용자**로서 블로그 글에 좋아요를 누르고 싶다.

- **관련 API**: `POST /api/v1/blogs/{id}/like`, `DELETE /api/v1/blogs/{id}/like`

### US-BL07: 블로거 구독
**로그인한 사용자**로서 마음에 드는 블로거를 구독하여 새 글 알림을 받고 싶다.

- **결과**: 구독 저장, 이후 해당 블로거 새 글 알림 수신
- **관련 API**: `POST /api/v1/blogs/{id}/subscribe`, `DELETE /api/v1/blogs/{id}/subscribe`

## PUSH 도메인

### US-P01: FCM 토큰 등록
**로그인한 사용자**로서 내 기기에 푸시 알림을 받기 위해 FCM 토큰을 등록하고 싶다.

- **트리거**: 앱 로그인 후 자동 처리 (Flutter FCM 초기화)
- **결과**: FCM 토큰이 서버에 저장됨
- **관련 API**: `POST /api/v1/push/tokens`

### US-P02: FCM 토큰 갱신
**사용자**로서 FCM 토큰이 만료/갱신될 때 서버의 토큰도 업데이트하고 싶다.

- **트리거**: Flutter FCM 토큰 갱신 콜백
- **관련 API**: `PUT /api/v1/push/tokens/{id}`

### US-P03: FCM 토큰 삭제 (로그아웃)
**사용자**로서 로그아웃 시 해당 기기로 알림이 오지 않도록 하고 싶다.

- **관련 API**: `DELETE /api/v1/push/tokens/{token}`

### US-P04: 알림 목록 조회
**로그인한 사용자**로서 받은 모든 알림을 목록으로 확인하고 싶다.

- **트리거**: 앱 알림 탭 진입
- **결과**: 알림 목록 (제목, 내용, 유형, 읽음 여부, 시간 포함)
- **관련 API**: `GET /api/v1/push/notifications`

### US-P05: 읽지 않은 알림 수 확인
**로그인한 사용자**로서 읽지 않은 알림이 몇 개인지 앱 뱃지로 확인하고 싶다.

- **트리거**: 앱 로드 시 또는 주기적 폴링
- **관련 API**: `GET /api/v1/push/notifications/unread/count`

### US-P06: 알림 읽음 처리
**사용자**로서 특정 알림을 읽음으로 표시하고 싶다.

- **관련 API**: `PUT /api/v1/push/notifications/{id}/read`

### US-P07: 전체 알림 읽음 처리
**사용자**로서 모든 알림을 한 번에 읽음으로 표시하고 싶다.

- **관련 API**: `PUT /api/v1/push/notifications/read-all`

### US-P08: 알림 삭제
**사용자**로서 불필요한 알림을 삭제하고 싶다.

- **관련 API**: `DELETE /api/v1/push/notifications/{id}`

## 우선순위 매트릭스

| 우선순위 | 유저 스토리 | 이유 |
|---------|------------|------|
| P0 (필수) | US-A01, US-A02, US-A03 | 인증 없이 다른 기능 불가 |
| P0 (필수) | US-C01, US-C03 | 핵심 채팅 기능 |
| P0 (필수) | US-B01, US-B02 | 게시판 핵심 |
| P1 (중요) | US-BL04, US-BL07 | 블로그 구독 피드 |
| P1 (중요) | US-P01, US-P04 | 알림 기본 기능 |
| P2 (선택) | US-B06, US-B10 | 편의 기능 |
| P1 (중요) | US-CS01, US-CS02 | 앱용 시뮬레이터 대시보드 |

## COIN SIMULATOR 도메인

### US-CS01: 시뮬레이터 대시보드 조회
**로그인한 사용자**로서 코인 시뮬레이터 상태와 거래 현황을 조회하고 싶다.

- **트리거**: 앱 `coint simulator` 화면 진입
- **결과**: 상태, 자산, 포지션, 최근 거래, 권한 정보 반환
- **관련 API**: `GET /api/v1/coin-simulator/dashboard`

### US-CS02: superuser 시뮬레이터 제어
**superuser**로서 시뮬레이터를 시작/중지하고 설정을 변경하고 싶다.

- **트리거**: 앱에서 Start/Stop 또는 설정 저장 버튼 탭
- **제약**: 일반 사용자는 읽기 전용
- **관련 API**: `POST /api/v1/coin-simulator/start`, `POST /api/v1/coin-simulator/stop`, `PUT /api/v1/coin-simulator/settings`
